## 推奨サイズ

- 縦長: 3:4 比率（例: 480×640px）
- フォーマット: PNG（透過あり推奨）または JPG
- 容量: 1枚あたり 500KB 以内推奨

## 生成AIへの参考プロンプト（例: セラフィナ）

```
anime style, fantasy RPG character, female knight,
blonde hair, golden armor, holy sword, light magic effects,
full body portrait, white background, transparent background,
high quality illustration, game card art
```

## 注意

- 画像がない場合は絵文字が表示されます（そのままでも遊べます）
- ファイル名は半角英数字・小文字で正確に入力してください
